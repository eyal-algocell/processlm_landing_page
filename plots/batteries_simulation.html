⚙ Model Equations & Parameters

SEI Growth (Tafel kinetics)
dδ_SEI/dt = k_SEI · exp(-E_a/RT)
R_SEI = δ_SEI / κ_SEI
// δ₀ = 5 nm, k_SEI = 1.5×10⁻¹⁶ m/s
Capacity Fade (LLI + LAM)
Q(n) = Q₀ · (1 − α·√n − β·n)
SOH = Q(n) / Q₀ · 100
// α=0.0018, β=0.000045
Voltage Model (SPM)
V(t) = U_OCV(SOC) − I·R_int
R_int = R₀ + R_SEI(n)
// R₀ = 12 mΩ, NMC cathode
Thermal Model (Lumped)
dT/dt = (Q_gen − Q_cool) / (m·Cp)
Q_gen = I²·R_int
// h=5 W/m²K, T_amb=25°C
OCV Curve (NMC)
U_OCV = 3.4 + 1.1·SOC
− 0.35·SOC² + 0.15·SOC³
// Fitted to NMC811 data
Noise Model
y_meas = y_true + ε
ε ~ N(0, σ²)
// σ_V=2mV, σ_T=0.3°C, σ_SOC=0.5%
